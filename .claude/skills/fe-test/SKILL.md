# FE TEST
작업한 코드들에 대해, 테스트 코드를 작성합니다. 

## 세부내용
- test 코드는 `frontend/test/` 아래에 작성합니다.
  - 필요하다면 하위 디렉토리를 새로 만들어 진행합니다.
- 테스트 코드를 작성하고, test 를 진행합니다. 
  - 테스트 결과는 ./.claude/docs/test.md 에 작성합니다.
  - FE/도메인/테스트 식으로 작성하고 feature 넘버를 작성합니다.
- 테스트 결과를 [] 에 체크박스로 표현합니다.
